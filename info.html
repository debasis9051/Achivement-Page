Achivement Page

Achivement Page Application provides colleges or institution to auto generate campusing posters,for those who have been successfully placed in a company. Students data are also collected and stored in database to generate Student profile which are available on Student database page.User can delete and student to database.

How to use the app

  • Step 1: Click on Student Database. Sign up with Email ID and Password
  • Step 2: Fill up the form with Company name and Six Student Data
  • Step 3: Click on Generate/Upload Student data ! This will GENERATE a corresponding Achievement PDF and will upload Student details to the Database
  • Step 4: Click on Preview or Download for downloading the PDF
  • Step 5: Click on student database ! It will show the available student profile for that user

Features !

  • Progressive Web App! Users can install and use it offline
  • Responsive design with Dark Theme and Light theme
  • Account authetication and Student Database

Technology!

Achievement Page uses a some of open source libraries to function properly:

  • [HTML5] - Frontend
  • [JSPDf] - Backend for Generating PDFs
  • [CSS3] - Frontend
  • [Bootstrap] - Frontend
  • [JAVASCRIPT] - Client Side
  • [Font Awesome] - Icons
